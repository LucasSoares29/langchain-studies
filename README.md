## Instruções  
Criar um arquivo .env com as chaves necessárias para acessar modelos de LLM e colocar na raiz do repositório.
Exemplo:

OPENAI_API_KEY = inserir-chave-aqui  
LANGCHAIN_API_KEY = inserir-chave-aqui
